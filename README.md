# LaTeX Editor

A local Overleaf-style split-view LaTeX editor, inspired by [OpenPrism](https://github.com/OpenDCAI/OpenPrism).

Write LaTeX on the left, see the compiled PDF on the right. Light/Dark mode. No subscription needed.

## Requirements

- Python 3
- MacTeX or BasicTeX (`brew install --cask basictex`)

## Run

```bash
bash run.sh
```

Then open http://localhost:8080
